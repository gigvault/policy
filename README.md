# GigVault policy

Policy Engine Service

## Features

- RESTful API
- Health and readiness endpoints
- Structured logging
- Configuration management

## API Endpoints

- `GET /health` - Health check
- `GET /ready` - Readiness check
- `GET /api/v1/status` - Service status

## Development

```bash
# Build
make build

# Run tests
make test

# Run locally
make run-local
```
